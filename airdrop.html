Token Airdrop

Send an ERC-20 token from a single source wallet to a list of recipient wallets, one transfer per address. Runs entirely in your browser against the Robinhood Chain RPC.

Robinhood Chain
⚠️ Private-key safety. The sender key you paste is used only in this browser tab to sign transfers and is never sent to any server. Only run this on a machine you trust, prefer a dedicated airdrop wallet, and clear the field when you're done.
